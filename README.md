## ⚙️ INSTALLATION GUIDE

### 🚧 1. Clone Project
```bash
git clone https://github.com/Luthfan-Rofian/manajemen-sdm-darululum.git
cd manajemen-sdm-darululum
```

### 📦 2. Install Dependencies
```bash
composer install
npm install
```

### 🔑 3. Setup Environment
```bash
cp .env.example .env
php artisan key:generate
```

### 🗄️ 4. Database Configuration

Edit `.env`:

```
DB_DATABASE=your_database
DB_USERNAME=your_username
DB_PASSWORD=your_password
```

### 🔄 5. Run Migration
```bash
php artisan migrate
```

### ▶️ 6. Start Server
```bash
php artisan serve
```

🌐 Akses aplikasi:  
http://localhost:8000
